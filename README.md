# Decimal Place Value Visualizer

An interactive web application that visualizes how numbers work on the right side of the decimal point in the place value system, showing both decimal notation and fractional representations with beautiful animations.

## Features

- **Interactive Input**: Type any decimal number and see real-time visualizations
- **Place Value Chart**: Visual grid showing each decimal place (tenths, hundredths, thousandths, etc.)
- **Fractional Breakdown**: See each digit displayed as a fraction (e.g., 3 in thousandths place = 3/1000)
- **Expanded Form**: View the sum of all fractional components
- **Animations**: 
  - Digits slide and fade into their place value positions
  - Sequential reveal from left to right
  - Highlight animations on active place values
  - Smooth fractional conversion animations
- **Auto-Demo Mode**: Watch animated examples cycle through automatically

## Usage

Simply open `index.html` in your web browser. No build process or dependencies required!

1. Enter a decimal number in the input field (e.g., `0.123`, `0.56789`)
2. Watch as the place values, fractions, and expanded form update in real-time
3. Click "Auto Demo" to see animated examples
